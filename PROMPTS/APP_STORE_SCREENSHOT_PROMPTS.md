## Galeri Temizleyici - Mağaza Ekran Görüntüsü Promptları

Bu dosya, Galeri Temizleyici uygulamasının App Store ve Google Play Store'da kullanılacak ekran görüntüleri için AI görsel oluşturma promptlarını içerir. Her özellik (Swipe, Blur, Duplicate) için kullanıcının dikkatini çekecek şekilde tasarlanmıştır.

## 📱 1. SWIPE ÖZELLİĞİ - Ana Ekran Görüntüsü

### Prompt:

```
Create a modern, premium mobile app screenshot showcasing a photo gallery cleaner app with a Tinder-like swipe interface.

VISUAL COMPOSITION:
- iPhone 15 Pro Max frame with notch and Dynamic Island visible
- Clean, minimalist Material Design 3 interface
- Subtle gradient background (slate-blue to light gray, top-left to bottom-right)

- Two smaller photo cards stacked behind the main card, creating depth with subtle shadows
- Swipe gesture indicators: Green checkmark icon on the right side (emerald green #22C55E) with "TUT" text, Red X icon on the left side (red #EF4444) with "SİL" text

- Primary: Slate/indigo blend (#334155)
- Keep/Accept: Emerald green (#22C55E)
- Delete: Red (#EF4444)
- Background: Light gradient with white and subtle blue tints
- Cards: White with soft shadows
- Text: Dark gray (#1F2937) for primary text, medium gray (#6B7280) for secondary

- Professional, clean, modern
- Trustworthy and intuitive
- Premium feel with subtle animations suggested

- Bright, well-lit interface

TECHNICAL SPECS:

## 🔍 2. BLUR ÖZELLİĞİ - Bulanık Fotoğraf Tespiti ve Yer Kazanma

### Prompt:

```

VISUAL COMPOSITION:

- Clean, minimalist Material Design 3 interface

- Split-screen or before/after layout showing the problem and solution
- Upper section (70% of screen): Grid view with MANY blurry photos, each showing file size below
- Lower section (30% of screen): Trash bin with space saved indicator and success message
- Visual flow: Blurry photos at top → Arrow/direction indicator → Trash bin at bottom with saved space

UPPER SECTION - BLURRY PHOTOS GRID:
- Grid layout: 4-5 columns showing 12-15 blurry photo thumbnails
- Each blurry photo thumbnail should be clearly out of focus, motion blurred, or pixelated
- EVERY blurry photo MUST have a file size label directly below it showing high MB values:
  - Examples: "267.9 MB", "189.3 MB", "324.7 MB", "156.2 MB", "298.4 MB", "412.8 MB"

  - Font: Bold, 11-12pt, dark gray or red color

- Each photo should have:
  - Red border (2-3px width) indicating blurry status
  - Red "BLURLU" badge or overlay in corner

  - Blur effect visible (motion blur, out of focus, pixelation)

- Rounded corners: 12px on all thumbnails

- Trash bin: Gray (#6B7280) or red accent
- Success container: Light green (#D1FAE5) with green border

TYPOGRAPHY:
- Modern, clean sans-serif font (SF Pro Display or Inter)
- File sizes: Bold, 11-12pt, dark gray/red
- Space saved: Extra bold, 24-28pt, emerald green
- Labels: Medium weight, 14-16pt, medium gray
- Turkish language text throughout
- Clear visual hierarchy with varying font sizes and weights

MOOD & STYLE:
- Professional and solution-oriented
- Shows clear value proposition (space savings)
- Before/after narrative (problem → solution)
- Trustworthy and efficient
- Modern and sleek
- High contrast for important information (file sizes, space saved)
- Visual satisfaction (seeing space being freed up)

- Resolution: 1290 x 2796 pixels
- Grid layout: 4-5 columns with 6-8px spacing
- Photo thumbnails: Rounded corners (12px radius)
- File size containers: Rounded corners (8px radius), subtle shadow
- Cards: 16-24px border radius
- Subtle shadows on all elevated elements
