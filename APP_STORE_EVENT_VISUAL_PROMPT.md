# App Store Event Visual - AI Prompt

## Event Details

- **Event Name**: Start Fresh: Clean Gallery
- **Theme**: New Year 2025 - Fresh Start
- **App**: Gallery Cleaner - Photo Gallery Organization App

---

## Main Visual Prompt (English)

```
Create a modern, premium App Store event banner for a photo gallery cleaning app.

**Composition:**
- Aspect ratio: 16:9 (landscape) or 9:16 (portrait) for App Store featured event
- Modern, clean design with gradient backgrounds
- Professional and inviting aesthetic

**Visual Elements:**
1. **Main Focus**: A clean, organized photo gallery interface mockup showing:
   - Before/After comparison: cluttered photos transitioning to organized, clean gallery
   - Swipe gesture indicators (left = delete, right = keep) with subtle motion lines
   - Duplicate photos being removed with visual effects
   - Blurry photos fading out

2. **New Year Theme Integration:**
   - Subtle 2025 numerals or "New Year" text integrated elegantly
   - Fresh, clean color palette: soft blues, greens, whites with accent colors
   - Sparkle effects or light rays suggesting a fresh start
   - Calendar or clock icon transitioning from December to January

3. **App Features Visualization:**
   - Photo cards stacked in a deck (Tinder-style swipe interface)
   - Storage space indicator showing "Space Freed" with upward arrow
   - Organized album folders/icons
   - Clean, minimalist UI elements

4. **Color Scheme:**
   - Primary: Soft gradient from #4A90E2 (blue) to #50C878 (green)
   - Background: Clean white (#FFFFFF) transitioning to light gray (#F5F5F5)
   - Accent: Vibrant accent color (#FF6B6B for delete, #51CF66 for keep)
   - Text: Dark gray (#2C3E50) for readability
   - Subtle shadows and depth for 3D effect

5. **Typography Style:**
   - Modern, clean sans-serif font (similar to Poppins)
   - Bold headline: "Start Fresh: Clean Gallery"
   - Subtitle: "New Year, New Gallery"
   - Feature highlights in smaller, readable text

6. **Visual Style:**
   - Flat design with subtle depth (soft shadows, gradients)
   - Smooth, rounded corners on UI elements
   - Clean lines and ample white space
   - Professional app mockup aesthetic
   - Light, airy feeling suggesting organization and clarity

7. **Additional Elements:**
   - Small icons: trash bin, duplicate symbol, blur indicator
   - Progress indicators showing cleaning process
   - Storage bar filling up with freed space
   - Subtle animation suggestions (motion blur, directional arrows)

**Mood & Tone:**
- Fresh, clean, organized
- Optimistic and forward-looking
- Professional yet approachable
- Modern and tech-forward
- Celebration of new beginnings

**Technical Requirements:**
- High resolution: 2048x1152px (16:9) or 1152x2048px (9:16)
- Suitable for App Store featured event placement
- Readable text at small sizes
- Works on both light and dark backgrounds
- Professional quality suitable for marketing

**Avoid:**
- Cluttered or busy design
- Too many competing elements
- Low contrast text
- Unprofessional or cartoonish style
- Overly complex animations or effects
```

## Usage Instructions

1. **For Midjourney/DALL-E/Stable Diffusion:**

   - Use the main English prompt above
   - Adjust aspect ratio parameter: `--ar 16:9` or `--ar 9:16`
   - Add style modifiers: `--style modern --quality high`

2. **For ChatGPT/DALL-E 3:**

   - Copy the simplified prompt or main prompt
   - Request: "Create an App Store event banner image..."
   - Specify dimensions: 2048x1152px or 1152x2048px

3. **For Adobe Firefly/Midjourney:**

   - Use the detailed prompt with all visual elements
   - Include technical requirements
   - Iterate with style refinements

4. **Post-Processing:**
   - Add app logo/branding if needed
   - Ensure text readability
   - Optimize for App Store requirements
   - Create variations for different placements
